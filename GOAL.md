# Project Goal

Build a [YOUR CHOICE — e.g. "simple personal expense tracker CLI tool" 
or "a small game" or literally "figure out something useful and build it"]

## Rules
- Work in small increments, one focused change per run
- Always run tests before committing
- Keep a STATE.md file updated with what you did and why
- Keep a DECISIONS.md file explaining your reasoning for major choices
- If something is broken, fix it before adding new features
